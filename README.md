# BanClass
课堂自主学习每一刻！  
Free classroom learning every moment!

> **免责声明**
> 本项目仅用于实现自主学习，将其用于不正当目的造成的一切后果与开发者无关。


## 特色

### 简单易用

-   免费安装；
-   布局清晰；
-   容易上手。

### 功能实用

-   支持对极域进行各种管控；
-   支持解除对Surf、Dino游戏的禁用。

### 缺点

-   只能在有自动还原的电脑上运行；
-   只支持在Windows8及更高版本的Windows系统上运行。
